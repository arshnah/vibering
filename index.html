What is a webring?

A webring is a collection of independent websites linked together in a loop. You start at one site, click "next", and you are taken to the next member's site, then the next, and so on, until you come back around to where you started. There is no ranking, no tracking, and no algorithm. Just a list of people's own websites, connected by hand.

How to join

This ring is invite-only. You cannot add yourself. A current member has to vouch for you, and an admin has to merge you in.

Step 1. Add your entry to the end of webring.json:

{
  "name": "your-handle",
  "gh": "your-github-username",
  "url": "https://your-site.com",
  "invitedBy": "member-who-invited-you"
}

Step 2. Put the ring navigation on your own site (replace your-handle with the name you used above):

<!-- loopback webring -->
<a href="https://arshnah.github.io/loopback/go.html?from=your-handle&dir=prev">prev</a> |
<a href="https://arshnah.github.io/loopback/random/">random</a> |
<a href="https://arshnah.github.io/loopback/">loopback</a> |
<a href="https://arshnah.github.io/loopback/go.html?from=your-handle&dir=next">next</a>

Step 3. Open a pull request and ping the member who invited you. They approve it with their own GitHub account, then an admin merges it in. Full instructions are in JOIN.md.
